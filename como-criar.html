jseckler

Como criar sua página pessoal no site da arco

(sim, vale também para você estudante)

Mande um email para jseckler arroba arco.coop.br informando nome de usuário e senha desejados. Suponha para o resto dessas instruções que o usuário escolhido foi usuario. A página ficará diponível em arco.coop.br/~usuario

A criação de usuário vai gerar uma pasta "home" (/home/usuario/) e, dentro dela, uma pasta www (/home/usuario/www/). Tudo que estiver dentro da pasta www está disponível na world wide web. O que estiver na sua pasta home mas fora da pasta www só é acessível por você e pelo administrador do servidor (no momento, eu).

Se quiser, crie uma página html e chame-a de index.html. Se não quiser, sua página pessoal mostrará simplesmente uma lista de arquivos armazenados, um pouco como essa daqui. Um jeito fácil de criar uma página html é escrever um documento no google docs e exportar como html (arquivo -> fazer o download -> página da Web)


No linux ou mac

Se ainda não estiver instalado, instale o programa scp

(em distribuições de linux baseadas em debian, faça $ sudo apt install scp)

Num terminal, copie sua página inicial para o servidor:

$ scp index.html usuario@arco.coop.br:~/www/

Copie uma pasta qualquer para o servidor:

$ scp -r pasta usuario@arco.coop.br:~/www/


No windows

Opção 1 - Criar um "Local de rede"

Já tivemos problemas com permissão de escrita usando esse método. Qualquer ajuda é bem vinda!

- Abra o windows explorer
-> meu computador -> botão direito -> adicionar local de rede
- Local de rede personalizado
- Endereço: ftp://arco.coop.br
- Especifique usuário e senha
- Dê um nome para este local de rede
- Concluir

Feito o procedimento acima, sua pasta "home" no servidor aparece como uma pasta para você dentro do windows explorer. Dentro dela, a pasta www, que corresponde à sua página pessoal na internet. Coloque seu arquivo index.html lá dentro e o que mais quiser.

Opção 2 - ftp pelo prompt de comando

Abra o prompt de comando (tecla windows + R, depois digite "cmd" e enter). Digite ftp arco.coop.br. Especifique usuário e senha. Aqui você tem acesso a alguns comandos:

  • put arquivo para fazer upload de um arquivo
  • get arquivo para fazer download de um arquivo
  • pwd para imprimir o nome da pasta em que você se encontra
  • cd diretorio para mudar de diretório
  • ls para listar os arquivos do diretório em que você está

Os nomes de arquivo usados são relativos ao diretório em que você se encontra quando invocou o comando ftp. Se precisar, especifique o caminho inteiro do arquivo (por exemplo: C:\Users\fulano\Downloads\arquivo). Outra opção é se mudar para o diretório antes de invocar o ftp (use, por exemplo, cd C:\Users\fulano\Downloads ou mesmo, talvez, cd Downloads).

Opção 3 - FileZilla

Instale o programa FileZilla e execute-o. No campo "Host", coloque arco.coop.br. Preencha seu usuário e senha e deixe o campo "Porta" em branco (estamos usando a porta padrão, que é a 21). Se tudo der certo, à esquerda você vai ver e manipular os arquivos na sua máquina e à direita os arquivos no servidor. Se Você pode começar a arrastar as coisas de um lado para o outro.

Acessando um terminal no servidor

Pelo linux e mac, basta ter instalado o ssh (se você instalou o scp, você já deve ter instalado o ssh também). Daí é só mandar um ssh usuario@arco.coop.br e fornecer o usuário e senha. No windows, um cliente de ssh bastante usado é o putty. Nas configurações, coloque arco.coop.br como host. Se ele perguntar, estamos usando a porta 22 para ssh.
